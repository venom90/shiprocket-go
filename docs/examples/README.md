# Examples

These examples are compileable Go programs. Set your environment and run them with:

```bash
go run ./docs/examples/<name>
```

Common environment variables:

- `SHIPROCKET_BASE_URL`
- `SHIPROCKET_TOKEN`
- `SHIPROCKET_EMAIL`
- `SHIPROCKET_PASSWORD`
- operation-specific IDs such as `SHIPROCKET_ORDER_ID`, `SHIPROCKET_SHIPMENT_ID`, or `SHIPROCKET_AWB`
