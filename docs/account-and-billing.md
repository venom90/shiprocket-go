# Account And Billing

## Covered operations

- Wallet balance
- Account statement
- Billing discrepancy
- File import result checks

## Operational use

- Wallet balance: preflight balance visibility before booking operations.
- Statement: operational reconciliation and support workflows.
- Billing discrepancy: finance or support review.
- Import result checks: follow-up after order, product, or listing file imports.

These calls are read-only and are the safest candidates for optional live smoke tests.
